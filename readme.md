## picoponicscan 
Sensor monitor and control firmware for an economical and scalable hydroponic system using an RP2040 pico and standard off the shelf sensors and components.
 *  air temp
 *  water temp
 *  humidity
 *  VOC
 *  CO2
 *  Soil H2O
 *  TDS
 *  EC
